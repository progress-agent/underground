UnderGround (MVP)

3D Tube network visualiser. Orbit: drag • Pan: right-drag/shift-drag • Zoom: wheel.

Data: TfL public endpoints (no auth). Cached for stability; works offline after first load.

V: Victoria stations • L: labels • F: focus Victoria • A: focus all visible lines

Tip: start focused on a line with ?focus=victoria (or any line id).

Time scale: 8×

Vertical exaggeration: 3.0×

Horizontal scale: 1.00×

Clears saved prefs + TfL cache.
